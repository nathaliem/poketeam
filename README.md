# Poketeam

A small project to play around with Angular 2+.<br />
Add Pokémon to your team and re-arrange them in the desired order.

Uses:
* Components
* Input, Output, Viewchild, etc
* Services
* Routing
* Observables
* Custom pipes
* EventEmitters
* HttpClient
* External libraries (Drag 'n Drop & Bootstrap)
* ...
